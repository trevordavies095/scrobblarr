Getting Started

How to obtain and configure Last.fm API credentials

1. Create a Last.fm API Account

Visit Last.fm API Account Creation and create a new API account.

2. Configure Environment Variables

Add the following to your .env file: 

LASTFM_API_KEY=your_api_key_here
LASTFM_API_SECRET=your_api_secret_here
LASTFM_USERNAME=your_lastfm_username
SYNC_FREQUENCY=hourly  # or daily, manual

3. Restart the Application

After updating the .env file, restart the application for changes to take effect: 

python manage.py runserver

4. Test Your Connection

Once configured, use the "Test Connection" button above to verify your credentials.
